**最後更新：2026-04-14**

## 一、投資風險說明

### 1.1 一般性風險警告

**投資有風險，入市需謹慎。**

CFPAI（Computational Finance Planning AI）是一套計算金融規劃系統，其輸出結果（包括但不限於資產權重、風險信號、路徑分析、回測績效）**僅供研究與參考用途**，不構成任何形式的投資建議、投資推薦或投資決策依據。

### 1.2 回測績效不代表未來表現

- 所有回測結果均基於**歷史數據**，歷史表現**不保證**也**不暗示**未來收益
- 回測可能存在**生存者偏差**（survivorship bias）、**前視偏差**（look-ahead bias）及**過度擬合**（overfitting）風險
- 白皮書及文檔中提及的 Sharpe ratio、年化收益率、最大回撤等指標，均為特定時間段、特定參數下的歷史模擬結果

### 1.3 模型風險

- CFPAI 的評分模型（包括 Maxwell Demon 門控機制）基於統計假設，這些假設在極端市場環境下可能失效
- UTM 調參結果可能過度擬合訓練數據，在樣本外（out-of-sample）環境中表現可能顯著下降
- LLM 情緒分析模組依賴第三方 AI 模型，其輸出可能存在幻覺（hallucination）、偏見或不準確

### 1.4 市場風險

- 金融市場受多重因素影響，包括但不限於：宏觀經濟變化、地緣政治事件、監管政策調整、流動性危機、黑天鵝事件
- 過往未曾發生的極端事件（如全球性疫情、戰爭、系統性金融危機）可能導致所有模型同時失效
- 個別資產可能因公司特定事件（財務造假、退市、破產）而發生超出模型預期的損失

### 1.5 槓桿與做空風險

- 使用槓桿（leverage）會**放大收益同時也放大損失**，最壞情況下可能導致全部本金損失甚至負債
- 做空（short selling）的潛在損失**理論上無上限**
- CFPAI 的槓桿安全檢查機制僅為輔助工具，不能完全防止極端情況下的損失

### 1.6 數據風險

- 市場數據來自第三方數據源（Stooq、Yahoo Finance 等），數據的準確性、完整性和時效性不由 CFPAI 保證
- 數據延遲、缺失或錯誤可能導致模型輸出不準確
- 免費數據源可能在未通知的情況下變更 API 或終止服務

## 三、LASA 相關聲明

LASA（Layered Accounting for Semantic Assets）是一套資金語義分類引擎：

- LASA 的分類結果**不構成**會計意見或稅務建議
- LASA 的異常檢測功能**不替代**專業的反洗錢（AML）或合規審查
- 使用者在進行稅務申報或合規報告時，應諮詢持牌會計師或法律顧問

## 四、LLM（大語言模型）相關聲明

CFPAI 集成了大語言模型（Claude）用於：

- 新聞情緒分析
- 策略決策解釋
- 異常資金語義分析

關於 LLM 輸出：

- LLM 可能產生不準確、不完整或具有誤導性的內容
- LLM 的輸出**不應**被視為專業金融分析或投資建議
- LLM 的知識有截止日期，可能不反映最新的市場狀況
- 使用者應獨立驗證 LLM 輸出的準確性

## 五、自動化投資顧問（Robo-Advisor）與金融科技聲明

### 5.1 CFPAI 不是自動化投資顧問（Robo-Advisor）

根據各主要司法管轄區的金融監管定義：

- **美國 SEC**：自動化投資顧問（Robo-Advisor）是指通過演算法為客戶提供個人化投資管理服務的註冊投資顧問（RIA），需在 SEC 或州政府註冊並接受監管
- **歐盟 MiFID II**：自動化投資建議屬於「投資建議」範疇，提供者需持有相應牌照
- **台灣金管會**：自動化投資顧問業務需經金管會核准，由合格的投信投顧業者提供

**CFPAI 不屬於上述任何一類。** CFPAI 是：

- 一套**開源研究工具**，不提供個人化投資管理
- 不接管使用者的資金或帳戶
- 不執行自動交易
- 不收取管理費或績效費
- 不持有任何金融服務牌照

### 5.2 金融科技（FinTech）與數字金融科普

為幫助使用者正確理解 CFPAI 的定位，以下說明金融科技領域的常見概念：

**量化投資工具的分類：**

| 類別 | 說明 | 需要牌照 | CFPAI 是否屬於 |
|------|------|----------|----------------|
| 自動化投資顧問（Robo-Advisor） | 為客戶提供個人化資產配置與自動再平衡 | 是 | **否** |
| 智能投顧平台 | 結合 AI 與人工顧問的混合服務 | 是 | **否** |
| 量化策略回測工具 | 提供歷史數據回測與策略驗證 | 通常不需要 | **是（僅此類）** |
| 交易信號生成器 | 生成買賣信號供使用者參考 | 視地區而定 | **部分功能** |
| 自動交易機器人 | 自動執行交易指令 | 視地區而定 | **否** |

**AI 在金融中的應用邊界：**

- **AI 可以做的**：數據分析、模式識別、風險量化、回測模擬、報告生成
- **AI 不能替代的**：投資判斷、風險承受能力評估、稅務規劃、合規審查、市場操縱偵測

**數字金融風險提示：**

1. **演算法偏見**：AI 模型可能在訓練數據中學到偏見，導致系統性錯誤
2. **黑箱風險**：複雜模型的決策過程可能難以解釋，CFPAI 通過 Maxwell Demon 解釋模組嘗試緩解此問題，但無法完全消除
3. **技術故障**：軟體可能存在 bug、數據源可能中斷、計算可能出錯
4. **過度依賴**：使用者可能因過度信任系統輸出而放鬆獨立判斷
5. **群體效應**：如果大量使用者使用相同策略，可能導致市場擁擠交易（crowded trade）
